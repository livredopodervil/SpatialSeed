# Materiais compartilhados por aparência — 0019g-b1

Associa cada `appearanceId` a um único `MeshStandardMaterial`.

Meta para 101 objetos equivalentes:

```text
uniqueGeometries: 1
uniqueMaterials: 1
uniqueTextures: 1
```

O realce emissivo individual fica desativado nesta etapa intermediária. Ele
será restaurado com `instanceColor` durante a integração do `InstancedMesh`.
