# SpatialSeed 0021b — apresentação e console

- `⛶` ou `F`: tela cheia real do navegador.
- `◉ Cena` ou `Tab`: somente a cena, sem sobreposição HTML.
- `Esc`: restaura a interface; em tela cheia, o navegador pode consumir o
  primeiro `Esc`.
- Tela cheia e somente-cena podem ser combinados.
- Cada painel recebe `⛶` para maximizar e `🗗` para restaurar.
- O console passa a ter janela própria.
- O scrub trava no eixo predominante:
  direita/cima aumenta; esquerda/baixo diminui.
