# Revisão da interface por comandos

A barra principal usa comandos canônicos para ferramentas, espaço, seleção, histórico, criação, pivô, duplicação, repetição e exclusão.

Condições operacionais normais, como excluir sem seleção, geram aviso no status. O painel vermelho permanece reservado a falhas internas.

A edição do pivô preserva o regime absoluto ou relativo ao objeto ativo.

O tamanho dos marcadores de vértices é configurável no painel de transformação.
